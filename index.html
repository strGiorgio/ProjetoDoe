Sobre o projeto

O projeto Doe inicialmente é uma ideia de como pessoas poderão fazer caridades doando pouco, mas mesmo assim ajudando milhares de pessoas, clínicas, hospitais, animais etc. Queremos fazer com que aqueles trocos que sobram de milhares de compras onlines todos os dias, sejam úteis para mais pessoas. A ideia do projeto é no futuro criar uma extensão que sempre que membros registrados fizerem alguma compra online, poderão escolher se querem doar os centavos que restam nas casas decimais.
